In details

Once textAnime.js is installed on your document, you can pass the following classes to any html text element. All four classes can be accumulated in the same element.

Class Requirement Description Code
txt-anime-{animation} Required This element triggers the scroll animation. You can choose from 31 different animations in the demo. Help yourself! <p class="txt-anime-slide-up">Exemple</p>
txt-delay-{value in milliseconds} Optional Adds an animation delay in miliseconds. Useful when you want to trigger animations one after the other! <p class="txt-anime-slide-up txt-delay-400">This text will appear after 400ms</p>
txt-split Optional Divides the text animation letter by letter rather than word by word, wrapping each word in a span so there's no issue with line breaks. <p class="txt-anime-slide-up txt-split">This text will appear letter after letter!</p>
txt-repeat Optional By default, animations are triggered only once. This option repeats the animation each time the text is visible in the viewport. <p class="txt-anime-slide-up txt-repeat">This text animation will be triggered each time I scroll back on it.</p>
